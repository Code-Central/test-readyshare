Before we continue, let's talk about the websites where these resources were grabbed from.

Open Game Art

"Open Game Art" is a website that many artists and musicians will post resources that game makers like yourself can add to their games.

When you are choosing a resource, make sure to look and see what they say you are allowed to use their work for.

Some will only let you use it for personal projects while others will also allow you to use it in games you intend to sell.

Always check the fine print! Otherwise, you could get into trouble for using something you weren't allowed to use...

Free PD

"Free PD" is a site filled with a lot of cool songs that are 100% free for you to use in any of your projects!

They are separated into different categories to help you find what you want a little easier.

There is only one problem with it, and that is that the file it gives you (MP3) is not usable in the Godot program. So we would need to use a tool like Online Audio Converter to convert the file from an MP3 to a WAV.

If you have trouble with this, ask an instructor for help on converting your file.

SoundBible

"Sound Bible" is a website filled with cool sound effects! Same as with "Open Game Art" make sure to check what the author of the sound effect says to make sure that you are allowed to use their sound file.

1001 Fonts

Using fancy fonts can help make your game look and feel fresh.

Just with other sites, not every font is free to use so make sure to read what the original creator says about how you are allowed to use their fonts...
